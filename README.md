# device-llrp-go
Edgex device service to handle LLRP Messages from 3rd Party RFID Readers.

## Build and deploy ##

### Prerequisites ###

 - Docker
 - Make
 - Docker-compose
 
### Build ###

```
git clone https://github.impcloud.net/RSP-Inventory-Suite/device-llrp-go.git
cd device-llrp-go
make build
```
